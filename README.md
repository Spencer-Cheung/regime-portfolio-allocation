# Market Regime Detection and Dynamic Portfolio Allocation

## Overview
This project explores how financial market regimes can be identified using historical market and how those regimes can be used to inform portfolio allocation decisions.

The goal is to classify market environments into distinct volatility or risk regimes (such as low, medium, and high volatility) and evaluate whether these regimes can improve portfolio positioning compared to a static investment strategy.

This project was developed as part of the **University of Michigan Master of Applied Data Science (MADS) Capstone (SIADS 699)**.
